## 课程目标
- 介绍 Hugging Face Agents 的功能和工作原理
- 对比 Hugging Face Agents 和 LangChain 的区别
- 演示如何使用 Agents 完成复杂的 AI 任务

## 核心知识点
- Agents 基本架构
- 使用大语言模型进行决策
- 自动选择和调用合适的模型
- 支持 run 和 chat 两种模式
- Agents 使用方法
  - agent.run() 命令的使用
  - agent.chat() 的特点和应用
- 工具和模型的调用方式
- 运行模式对比
  - Run 模式：无记忆，适合多任务连续执行
  - Chat 模式：有记忆，适合单任务交互
- 清除记忆和重置会话的方法

## 扩展知识（关键词解释）
- Transformer Agents：Hugging Face 推出的自动化 AI 工具，能够理解用户指令并自动选择合适的模型来完成任务，简化了 AI 应用开发流程。
- Agent Memory：AI Agent 在运行过程中保存的上下文信息，使其能够在多轮对话中保持连贯性和任务连续性。
- Model Hub：Hugging Face 提供的模型托管平台，包含各种预训练模型，支持 Agents 调用不同类型的 AI 模型完成特定任务。
- API Key：访问 AI 服务所需的认证密钥，包括 Hugging Face 和 OpenAI 的访问凭证，用于确保服务的安全使用。
- Task Decomposition：Agent 将复杂任务分解为可执行步骤的过程，包括理解任务、选择工具和执行操作等环节。
- Tool Integration：不同 AI 工具和模型的集成方式，包括模型调用、API 访问和代码执行等多种形式。
