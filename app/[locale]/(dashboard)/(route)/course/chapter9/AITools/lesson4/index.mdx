## 课程目标
- 介绍 Hugging Face 模型库的分类和使用方法
- 演示如何在实际场景中应用不同类型的模型
- 通过实例帮助学习者理解和使用自然语言处理模型

- Hugging Face 模型分类
  - 计算机视觉（8类）
  - 自然语言处理（11类）

  - 表格处理
  - 多模态处理

  - 自定义模型参数
  - 结果解析和应用

## 扩展知识（关键词解释）
- Task Page：Hugging Face 提供的任务导向页面，帮助用户根据具体需求快速找到合适的模型、数据集和示例代码。
- Pipeline：Transformers 库中的核心功能，将复杂的模型调用过程简化为单一接口，使用户能够快速实现各种 AI 任务。
- Zero-shot Classification：一种无需预先训练的分类方法，允许模型对未见过的类别进行分类，提高了模型的灵活性和实用性。
- Sentiment Analysis：情感分析技术，通过 AI 模型自动判断文本表达的情感倾向，广泛应用于用户评论分析和舆情监测。
- Information Extraction：从非结构化文本中提取结构化信息的技术，能够自动识别并提取文本中的关键信息，如人名、地点、组织等。
- Model Selection：在 Hugging Face 平台上选择合适模型的过程，包括根据任务类型、性能要求和资源限制等因素进行筛选。
