如何创建第一个 Next.js 项目。需在终端运行 npx create-next-app，建议使用版本 13.4 以避免课程中的问题。创建项目时，需回答以下问题：1) 项目名称，示例为 next-app；2) 是否使用 TypeScript，默认选是；3) 是否使用 ESLint 代码分析工具，默认选是；4) 是否使用 Tailwind CSS，默认选是；5) 是否使用源目录，选否；6) 是否使用新 App Router，选是（课程使用新 App Router，而非旧 Pages Router）；7) 是否自定义默认导入别名，选否。安装依赖（包括 React、Next.js、TypeScript 等）后，进入项目文件夹，运行 npm run dev 启动开发服务器，访问端口 3000 确认项目运行成功。
