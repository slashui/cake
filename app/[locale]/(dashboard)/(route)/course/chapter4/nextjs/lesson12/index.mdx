本节介绍了 Tailwind CSS，一个基于工具类（utility classes）的流行 CSS 框架，通过类如 flex、pt-4（上边距 4）、text-center 等组合样式应用。Tailwind 提供大量小型工具类（如 p-5 表示 5 单位内边距，mx-5 水平外边距，text-xl 字体大小，bg-sky-400 背景色等），无需记忆，随实践逐渐熟悉。颜色基于默认调色板（如 sky-50 至 sky-950），可自定义品牌颜色。支持伪选择器（如 hover:bg-sky-600 鼠标悬停时更改背景色）。在项目中，替换 CSS 模块为 Tailwind，直接在组件内应用工具类（如 p-5 my-5 bg-sky-400 text-white text-xl hover:bg-sky-600），无需单独 CSS 文件，减少文件切换。Tailwind 优点包括：1) 样式与组件代码共存，便于维护；2) 构建时仅包含用到的工具类，生成更小 CSS bundle；3) 删除组件无需清理 CSS 文件，优于 CSS 模块需手动删除未用类。缺点是复杂标记可能导致类名过多，代码可读性稍差，但随习惯改善。Tailwind 因高市场需求，建议学习以提升职业竞争力。
