## 一、创建 GitHub 仓库
1. 登录 GitHub 账号
2. 点击右上角的 "+" 号，选择 "New repository"
3. 填写仓库信息：
   - 输入仓库名称
   - 选择是否公开（Public）或私有（Private）
   - 不要勾选任何初始化选项（如 README、.gitignore 等）
4. 点击 "Create repository"
## 二、配置 SSH 密钥
## 1. 生成 SSH 密钥对
```bash
# 生成 SSH 密钥，替换 your_email@example.com 为你的邮箱
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

# 按回车接受默认文件位置（~/.ssh/id_rsa）
# 可以选择是否设置密码，如果不需要直接按回车
 ```

## 2. 查看公钥内容
```bash
# 显示公钥内容
cat ~/.ssh/id_rsa.pub
 ```

## 3. 添加 SSH 密钥到 GitHub
1. 复制公钥内容（从 ssh-rsa 开始到邮箱地址结束的全部内容）
2. 在 GitHub 网站：
   - 点击右上角头像
   - 选择 "Settings"
   - 左侧菜单找到 "SSH and GPG keys"
   - 点击 "New SSH key"
   - 给密钥起个名字（如：MacBook Pro）
   - 粘贴公钥内容
   - 点击 "Add SSH key"

## 三、本地仓库设置
## 1. 初始化本地仓库（如果是新项目）
```bash
# 在项目目录下初始化 Git 仓库
git init
 ```

## 2. 关联远程仓库
```bash
# 添加远程仓库，使用 SSH 地址
git remote add origin git@github.com:用户名/仓库名.git

# 验证远程仓库配置
git remote -v
 ```

## 3. 推送代码
```bash
# 首次推送，设置上游分支
git push -u origin main
 ```

## 四、验证 SSH 连接
```bash
# 测试 SSH 连接是否成功
ssh -T git@github.com
 ```

如果看到类似 "Hi username! You've successfully authenticated..." 的消息，说明 SSH 配置成功。
