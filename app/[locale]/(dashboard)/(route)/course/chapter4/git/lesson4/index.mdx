最受欢迎的远程仓库存储地是GitHub，它由微软拥有，并且可以免费使用。如果你还没有GitHub账号，可以去GitHub.com免费注册一个。注册之后，我们将使用GitHub Desktop应用在我们的电脑上运行GitHub。现在我想花一点时间解释一下GitHub和GitHub Desktop之间的区别，因为这可能会让人有点困惑。

Git才是真正的版本控制软件，它最初的设计是在命令行中运行的，而且很多开发者现在仍然使用这种方式。


GitHub是一个网站，你可以在上面托管你的Git仓库。你还可以与其他的用户协作，并发现其他的开源仓库。

## 步骤 1：访问 GitHub 官网
打开你的浏览器。
在地址栏输入 https://github.com，然后按回车键进入 GitHub 首页。
## 步骤 2：开始注册
在 GitHub 首页右上角，找到并点击 Sign up（注册）按钮。
你会看到一个注册表单，标题为“Create your account”（创建你的账号）。
## 步骤 3：填写注册信息
用户名 (Username):
输入一个独特的用户名。这是你在 GitHub 上的身份标识，建议使用字母、数字或短横线（-）的组合。
如果用户名已被占用，GitHub 会提示你更换。
示例：mygithubname
邮箱地址 (Email Address):
输入一个有效的电子邮件地址，用于接收验证邮件和通知。
示例：yourname@example.com
密码 (Password):
创建一个安全的密码，至少 8 个字符，建议包含字母、数字和特殊字符。
示例：MyPassw0rd!
点击 Continue（继续）按钮。
## 步骤 4：验证身份
GitHub 会要求你完成一个简单的验证步骤，以确保你不是机器人。
按照屏幕上的提示完成拼图或点击验证按钮。
## 步骤 5：选择计划
注册表单完成后，GitHub 会显示计划选择页面。
对于新手，推荐选择 Free（免费）计划，点击 Choose Free。
免费计划提供无限公共和私有仓库，足以满足个人学习和小型项目需求。
如果未来需要更多功能，可以随时升级到付费计划。
## 步骤 6：验证邮箱
注册完成后，GitHub 会向你提供的邮箱发送一封验证邮件。
打开你的邮箱，找到来自 GitHub 的邮件（主题可能类似于“Verify your email address”）。
点击邮件中的验证链接，完成邮箱验证。
如果未收到邮件，请检查垃圾邮件或垃圾箱文件夹。
## 步骤 7：完成注册
验证邮箱后，你会被重定向到 GitHub 的仪表板（Dashboard）。
恭喜！你已成功注册 GitHub 账号，可以开始探索和使用了！
