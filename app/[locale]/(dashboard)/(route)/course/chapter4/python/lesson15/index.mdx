本节讲解了在Python中如何通过循环遍历列表并逐个访问元素。

### for循环：

- 通过for item in numbers:结构，直接逐项获取列表[1, 2, 3, 4, 5]中的元素，无需手动管理索引。
- 每次迭代时，变量item自动更新为当前元素值（如第一次为1，第二次为2），代码简洁高效。

### while循环对比：

- 需额外声明索引变量（如i=0），通过while i < len(numbers):控制循环条件，并用numbers[i]访问元素，最后需手动递增i。
- 代码冗长且易出错，远不如for循环直观。

### Note：

- for循环是遍历列表的首选方式，无需处理索引和长度，可读性更强；
- while循环虽能实现相同功能，但需更多代码维护状态，适用于更复杂的循环逻辑场景。
