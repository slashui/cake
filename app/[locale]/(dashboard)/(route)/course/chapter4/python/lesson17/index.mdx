本节介绍了Python中的元组（tuple），它是一种类似于列表（list）的序列类型，但具有不可变性（immutable）。

### 元组与列表的区别：

- 列表用方括号 [] 定义（如 [1, 2, 3]），元组用圆括号 () 定义（如 (1, 2, 3)）。
- 元组一旦创建无法修改（如尝试 numbers[0] = 10 会报错 TypeError）。

### 元组的方法：

- 仅支持 count()（统计元素出现次数）和 index()（返回元素首次出现的索引）。
- 无 append()、insert() 等修改方法，确保数据安全。

### 适用场景：

当需要存储不应被意外修改的数据时（如配置项、常量集合），元组比列表更合适。
