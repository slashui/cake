本节讲Python里的运算符优先级，跟数学里一样，先算乘除后算加减。

通过一个例子（x = 10 + 3 * 2）说明为什么结果是16（先算3 * 2得6，再加10）。

还会展示怎么用括号改变优先级，比如(10 + 3) * 2得到26。简单例子让你明白运算顺序怎么影响结果。

### Note
运算符优先级很重要，乘除（*、/）比加减（+、-）优先级高，想改变顺序就用括号。

练习时可以试试不同表达式，比如5 + 2 * 3或(5 + 2) * 3，看看结果有啥区别，帮你更清楚优先级的效果。
