## 可以访问 unsplash 的 MCP

https://github.com/hellokaton/unsplash-mcp-server

Unsplash MCP Server提供了一个简单而强大的界面，通过MCP搜索和利用Unsplash的大量高质量图像集合。非常适合希望将Unsplash功能集成到其应用程序或Cursor编辑器工作流程中的开发人员。

## ✨特点

高级图像搜索：使用过滤器搜索Unsplash的大量照片库：
关键词相关性
配色方案
方向选项
自定义排序和分页

## 🔧先决条件

Python 3.9或更高版本
UV软件包管理器（pip install uv）
Unsplash API访问密钥（在Unsplash开发人员门户网站注册）
