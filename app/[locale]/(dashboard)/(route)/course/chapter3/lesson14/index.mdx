本节课介绍了如何在 Cursor 中配置和使用 Browser MCP，实现 AI 与浏览器的交互，使 AI 能够访问和处理网页内容。

## 配置步骤

1. **安装 Browser MCP**
   - 访问 [mcp.so](https://mcp.so) 网站
   - 搜索 "Browser MCP"
   - 查看文档和配置说明

2. **Chrome 插件安装**
   - 安装 Browser MCP Chrome 扩展
   - 在 Chrome 中启用插件
   - 点击连接按钮激活

3. **Cursor 配置**
   - 打开 Cursor 设置
   - 找到 MCP 设置选项
   - 在 Global MCP Server 中添加配置
   - 粘贴 Browser MCP 的配置 JSON

## 连接验证

1. **状态指示**
   - 黄点：未连接状态
   - 绿点：连接成功状态

2. **连接步骤**
   - 在 Chrome 中打开 MCP 插件
   - 点击连接按钮
   - 确认 Cursor 中 MCP 状态变为绿色

## 使用示例

1. **信息搜索**
   - AI 可以直接操作浏览器
   - 自动导航到相关网页
   - 提取所需信息

2. **实际案例**
   - 搜索 OpenAI 的最新 API 信息
