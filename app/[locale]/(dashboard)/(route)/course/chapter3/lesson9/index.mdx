本节课介绍了如何在 Cursor 中添加外部文档并使用网络搜索功能，以提高开发效率。通过实际案例展示了如何集成 JSONPlaceholder API 到 Todo 应用中。

## 核心功能

1. **文档集成功能**
   - 支持添加外部 API 文档
   - 可以在对话中直接引用文档内容
   - 无需手动阅读和理解文档细节

2. **Web 搜索功能**
   - 可搜索最新技术资讯
   - 自动总结搜索结果
   - 提供相关参考链接

## 实践案例：集成 JSONPlaceholder

1. **添加测试数据**
   - 使用 [JSONPlaceholder](https://jsonplaceholder.typicode.com/) 提供的免费 API
   - 获取模拟的 Todo 数据
   - 自动生成占位内容

2. **集成步骤**
   - 复制 API 文档地址
   - 在 Cursor 中添加为文档
   - 通过 AI 助手自动生成集成代码
   - 实现数据展示和交互功能

## 使用优势

- 无需手动阅读复杂文档
- AI 自动理解并实现 API 集成
- 提高开发效率
- 自动处理数据交互逻辑

## Web 搜索示例

- 搜索最新技术信息（如 Next.js 15 新特性）
- 获取实时更新的文档和教程
- 自动总结关键信息

## 扩展知识

1. **API 测试资源**
   - JSONPlaceholder 提供免费的 REST API
   - 支持多种数据类型（todos, posts, users 等）
   - 适合原型开发和测试

2. **最佳实践**
   - 在开发初期添加占位数据
   - 使用真实场景的测试数据
   - 保持用户界面的真实感

## 小贴士

- 优先使用文档集成功能处理已知 API
- 使用 Web 搜索获取最新技术信息
- 合理利用 AI 提供的代码建议
- 注意验证生成代码的正确性
