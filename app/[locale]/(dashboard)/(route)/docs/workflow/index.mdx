## 课程目标
- 介绍 Hugging Face 平台及其在 AI 领域的重要性

- 掌握使用 Transformers 库的基础知识

## 核心知识点
- Hugging Face 平台介绍
  - 成立于 2016 年，最初是聊天机器人公司
  - 2019 年开源 Transformers 库后快速发展
  - 目前托管超过 20 万个预训练模型和 1 万多个数据集
  - 被称为"AI 界的 GitHub"
- 平台主要组成部分
  - Models（模型库）
  - Datasets（数据集）
  - Spaces（应用展示空间）
- Transformers 库基础使用
  - 安装方法：pip install transformers
  - Pipeline 工具的使用
    - 三个核心步骤：预处理、模型应用、后处理
  - 实践示例
    - 情感分析示例
    - 文本生成示例
    - 零样本分类示例

## 扩展知识
- Image to Text：一种将图像转换为文字描述的 AI 技术，通过预训练的深度学习模型识别图像内容并生成相应的文本描述，是多模态 AI 应用的基础功能之一。
- Text to Speech：将文本转换为语音的技术，通过 AI 模型将书面文字转化为自然的语音输出，可用于生成各种语言和音色的语音内容。
- 系统 Prompt：在使用大语言模型时预先设定的指令或上下文，用于指导模型生成特定风格或格式的内容，是确保 AI 输出符合预期的关键要素。
- API 调用：通过网络请求访问远程 AI 服务的方法，包括请求格式、参数设置和响应处理，是构建 AI 应用的标准方式。
- 音频处理：对 AI 生成的语音数据进行处理的技术，包括文件格式转换、保存和播放等操作，是语音应用开发中的重要环节。
- 模型串联：将多个 AI 模型按照特定顺序组合使用的技术，通过处理流程的设计实现复杂的 AI 功能，是构建综合性 AI 应用的核心方法。
